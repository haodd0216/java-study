## 练习内容

菜单驱动的数字分析器，综合运用：
- switch 菜单分发
- while(true) + break 主循环
- for 循环（素数判断、阶乘、乘法表）
- while 循环（各位数字之和）
- if-else 条件判断

## 完成情况

| 功能 | 状态 | 说明 |
|------|------|------|
| 素数判断 | ✅ | 逻辑正确，三元表达式处理分支 |
| 阶乘计算 | ✅ | 标准 for 累乘 |
| 各位数字之和 | ✅ | while 取余累加 |
| 乘法表 | ✅ | 嵌套 for，支持指定行数 |
| 菜单循环 + 退出 | ✅ | while(true)+break |

## 亮点
- 用并行数组 `menuChoices` + `arguments` 模拟输入，设计简洁
- 素数判断用了一个循环 + 三目运算符的紧凑写法

## 改进点
- 素数判断：边缘情况 n≤1 无输出；建议用 boolean flag 或 sqrt(n) 优化
- 输出格式：中文前后建议加空格（"7 是素数"）
