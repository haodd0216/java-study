# 多态综合练习 — 通知系统

日期：2026-07-03
主题：Java / 面向对象编程 / 多态 / 综合练习

## 练习内容

构建了一个通知系统，综合运用了多态的全部知识点：

- 抽象类 `Notification`：定义通知的共同结构（sender, content, preview(), abstract send()）
- 接口 `Trackable`：定义可追踪能力（getChannelName(), getDeliveryStatus()）
- 三个子类 `EmailNotification`, `SMSNotification`, `PushNotification`：分别实现通知和追踪

## 验收结果

### 通过项

1. ✅ Notification[] 数组 — 向上转型（抽象类多态）
2. ✅ 遍历调用 preview() 和 send() — 动态绑定
3. ✅ instanceof + 模式匹配 — 安全向下转型，访问子类特有字段
4. ✅ Trackable[] 数组 — 接口多态
5. ✅ 理解抽象类不能 new
6. ✅ 理解父类引用不能调用子类特有方法

## 掌握的多态知识体系

- 向上转型（子类 → 父类/抽象类/接口）
- 向下转型（父类 → 子类，需要 instanceof 守卫）
- instanceof 运算符 + Java 16 模式匹配
- 动态绑定（方法看实际类型，字段看声明类型）
- 抽象类（不能 new，强制子类实现）
- 接口多态（一个对象多种引用视角）
