## 定位

变量与数据类型是 Java 程序中最核心的"存储单元"。前面学了程序的基本结构（类、main 方法），现在要回答的问题是：**程序运行时，数据存在哪里？怎么存？能存什么？** 在 Java 知识地图中属于 `语言基础` 分支，是第 3 个概念，位于 `基本语法与程序结构` 之后、`运算符与表达式` 之前。

## 类比

把 Java 程序想象成一个**仓库管理系统**：

- **变量** = 仓库里的**储物箱**，每个箱子贴着标签（变量名），方便你随时找到它
- **数据类型** = 箱子的**规格说明**，规定了这个箱子只能放什么东西：这是装螺丝的小盒（`int`），那是装文件的大柜（`String`），别的是只能开/关的开关（`boolean`）

Java 和某些语言（如 Python、JavaScript）不同——它的箱子是**订制的**：一旦你定了箱子规格，就不能往里面塞不匹配的东西。这叫**强类型语言**。

## 核心机制

Java 的数据类型分两大类：

### 8 种基本类型（Primitive Types）——直接存值

| 类型 | 大小 | 范围 | 例子 | 备注 |
|------|------|------|------|------|
| `byte` | 1 字节 | -128 ~ 127 | `byte b = 100;` | 极少用，IO 流中偶尔出现 |
| `short` | 2 字节 | -32768 ~ 32767 | `short s = 30000;` | 也很少用 |
| `int` | 4 字节 | -21亿 ~ 21亿 | `int i = 42;` | **默认整数类型** |
| `long` | 8 字节 | 极大 | `long l = 9999999999L;` | 需要加 `L` 后缀 |
| `float` | 4 字节 | 单精度 | `float f = 3.14f;` | **必须加 `f` 后缀** |
| `double` | 8 字节 | 双精度 | `double d = 3.14;` | **默认小数类型** |
| `char` | 2 字节 | 0 ~ 65535（Unicode） | `char c = 'A';` | 用**单引号** |
| `boolean` | JVM 决定 | `true` / `false` | `boolean ok = true;` | 只有两个值 |

### 引用类型（Reference Types）——存地址

`String`、数组、所有对象都属于引用类型。变量里存的是**对象在内存中的地址**，而不是对象本身。

```java
String name = "张三";   // name 存的是 "张三" 这个字符串对象的地址
int[] arr = {1, 2, 3};  // arr 存的是数组对象的地址
```

**关键区别**：基本类型变量直接存值，引用类型变量存的是"门牌号"。

### 变量的声明与赋值

```java
// 声明 + 赋值可以分开
int age;        // 声明：告诉 JVM "我要一个 int 类型的变量叫 age"
age = 25;       // 赋值：把 25 放进去

// 声明 + 赋值一步到位（推荐）
double price = 9.99;

// 用 var 自动推断类型（Java 10+，类型仍然是确定的）
var count = 10;      // 编译器推断为 int
var text = "hello";  // 编译器推断为 String
```

### 类型转换

- **自动转换（隐式）**：小范围 → 大范围，安全，自动完成
  ```java
  int i = 100;
  long l = i;      // int 自动转 long，没问题
  double d = i;    // int 自动转 double，100 → 100.0
  ```

- **强制转换（显式）**：大范围 → 小范围，可能丢失精度，需要手动写
  ```java
  double pi = 3.14159;
  int n = (int) pi;   // 强制转换，小数部分被砍掉，n = 3
  ```

### 常量：`final` 关键字

```java
final double TAX_RATE = 0.13;  // 一旦赋值就不能再改
// TAX_RATE = 0.15;  // 编译错误！
```

## 代码示例

```java
public class VariableDemo {
    public static void main(String[] args) {
        // ---- 基本类型 ----
        int studentCount = 45;               // 整数默认 int
        long worldPopulation = 8_000_000_000L; // long 要加 L
        double avgScore = 87.5;              // 小数默认 double
        float pi = 3.14f;                    // float 必须加 f
        char grade = 'A';                    // 字符用单引号
        boolean isPassed = true;             // 布尔只有 true/false

        // ---- 引用类型 ----
        String name = "李华";                 // 字符串用双引号
        int[] scores = {90, 85, 92};         // 数组

        // ---- 类型转换 ----
        double precise = studentCount;       // int → double，自动
        int rounded = (int) avgScore;        // double → int，强制，结果 = 87

        // ---- 常量 ----
        final int MAX_STUDENTS = 50;         // 常量命名习惯全大写

        // ---- 输出 ----
        System.out.println(name + " 的成绩是 " + grade);
        System.out.println("总分: " + (scores[0] + scores[1] + scores[2]));
    }
}
```

**代码走读：**

- `studentCount` 用 `int`，因为学生人数在 21 亿以内
- `worldPopulation` 用 `long` 并加 `L`，70 亿超出 int 范围
- `pi` 必须写 `f` 后缀，否则编译器把 `3.14` 当成 `double` 塞不进 `float`
- `grade` 用**单引号** `'A'`；`String` 用**双引号** `"李华"`——这是新手必踩的坑
- `precise` 自动得到 `45.0`；`rounded` 强制转换会**截断**（不是四舍五入），`87.5` 变成 `87`

## 常见误区

**误区一："我没赋值，变量应该有默认值吧？"**
- **局部变量**（方法里声明的）**没有默认值**，不赋值直接用，编译报错
- **成员变量**（类里声明的）有默认值（`int` 默认 0，`boolean` 默认 false，引用类型默认 null）
- 所以 `int x; System.out.println(x);` 写在 main 方法里会报错

**误区二："3.14 就是 float"**
- `3.14` 在 Java 里默认是 `double` 类型，要表示 `float` 必须写 `3.14f`
- 反过来 `long x = 100;` 不报错是因为 `100` 在 int 范围内，Java 自动帮你转了，但 `long x = 9999999999;` 就会报错，因为数字超出了 int 范围，必须写 `9999999999L`

**误区三："整数除法会给我小数结果"**
- `int a = 5; int b = 2; System.out.println(a / b);` 输出 `2`，不是 `2.5`
- Java 中整数除以整数，结果还是整数（截断），想得小数得把其中一个转成 `double`

## 快速总结

- Java 是强类型语言，每个变量必须先声明类型才能使用
- 8 种基本类型：byte、short、int、long、float、double、char、boolean
- 整数默认是 `int`，小数默认是 `double`
- `float` 字面量必须加 `f` 后缀，`long` 超大值必须加 `L` 后缀
- 基本类型存值，引用类型存地址
- 小范围 → 大范围自动转，反之需要强制转换（可能丢精度）
- 局部变量没有默认值，必须初始化
