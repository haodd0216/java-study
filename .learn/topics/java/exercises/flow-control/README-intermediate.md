## 需求：数字分析器

实现一个控制台菜单程序，功能如下：

### 菜单（switch 实现）

```
===== 数字分析器 =====
1. 判断素数
2. 计算阶乘
3. 各位数字之和
4. 打印乘法表（指定行数）
0. 退出
请选择:
```

### 功能说明

| 选项 | 功能 | 实现要点 |
|------|------|---------|
| 1 | 输入一个正整数，判断是否为素数 | for 循环 + if + break 提前退出 |
| 2 | 输入 n，计算 n! | for 循环累乘，注意 0! = 1 |
| 3 | 输入一个整数，求各位数字之和（如 123 → 6） | while 循环取余累加 |
| 4 | 输入行数 n（1-9），打印左下三角乘法表 | 嵌套 for，与 Task5 类似但行数可变 |
| 0 | 退出程序 | break 退出主循环 |
| 其他 | 提示"无效选项，请重新选择" | switch default |

### 主循环

- 用 `while (true)` 包住菜单，选 0 时 `break` 退出
- 用 `continue` 处理无效输入后回到菜单

## 输入模拟

因为控制台交互不方便，你**不需要写 Scanner**。直接在代码中用变量模拟输入即可：

```java
// 模拟输入：依次执行选项 1(判7)、2(算5!)、3(算123)、4(打印6行)、0(退出)
int[] menuChoices = {1, 2, 3, 4, 0};
int[] arguments = {7, 5, 123, 6};  // 对应每个选项需要的参数
```

用索引遍历 `menuChoices`，根据选项取对应的 argument。

## 覆盖技能

| 技能 | 使用位置 |
|------|---------|
| switch | 菜单分发 |
| while(true) + break | 主循环 + 退出 |
| for | 素数判断、阶乘、乘法表 |
| if-else | 素数判断、输入校验 |
| continue | (可选) 无效菜单项时跳过 |
| 嵌套循环 | 乘法表打印 |
