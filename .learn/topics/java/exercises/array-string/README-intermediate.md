# 数组与字符串 — 中级挑战

> **Topic:** java | **Path:** 语言基础/数组与字符串
> **Difficulty:** intermediate | **1 道综合题**

---

## 练习前须知

1. 在 `com/fuhao/arraystring/` 目录下创建 `ScoreManager.java`（包名 `com.fuhao.arraystring`）
2. 这是一个**学生成绩管理系统**，综合运用数组操作 + String 处理
3. 各功能写在同一个类里，用 switch + while 菜单驱动（参考 NumberAnalyzer 的模式）

---

## 需求：学生成绩管理系统

### 数据

```java
String[] names = {"张三", "李四", "王五", "赵六", "陈七"};
int[] scores = {85, 92, 58, 73, 66};
```

### 功能

| 选项 | 功能 | 涉及知识点 |
|------|------|-----------|
| 1 | **打印成绩单** — 用 `StringBuilder` 构建一张表格，格式见下方 | StringBuilder, String.format() |
| 2 | **统计信息** — 最高分、最低分、平均分（保留 1 位小数） | Arrays.sort(), 数组遍历 |
| 3 | **等级分布** — 统计各等级人数：A(≥90) B(≥80) C(≥70) D(≥60) F(<60) | 数组遍历, if-else |
| 4 | **查找学生** — 根据名字查找分数（`String.equals()`） | String 比较, 线性查找 |
| 5 | **添加学生** — 扩展数组，添加一名新生（用 `Arrays.copyOf()`） | Arrays.copyOf() |
| 0 | **退出** | break |

### 输出格式

**选项 1 的成绩单格式：**
```
===== 成绩单 =====
张三        85    B
李四        92    A
王五        58    F
赵六        73    C
陈七        66    D
==================
共 5 名学生
```

**选项 2 的统计信息格式：**
```
最高分: 92 (李四)
最低分: 58 (王五)
平均分: 74.8
```

**选项 3 的等级分布格式：**
```
A (90-100): 1 人
B (80-89):  1 人
C (70-79):  1 人
D (60-69):  2 人
F (0-59):   1 人
```

**选项 4 — 查找：**
```
请输入姓名: 王五
王五 → 58 分

请输入姓名: 刘八
未找到学生: 刘八
```

**选项 5 — 添加（扩展数组）：**
```
添加学生: 刘八, 分数: 88
添加成功！现有 6 名学生
打印成绩单确认:
...
```

## 覆盖技能总结

| 技能 | 使用场景 |
|------|---------|
| 数组声明与初始化 | names[], scores[] |
| 数组遍历 (for) | 打印成绩单、统计、查找 |
| Arrays.sort() | 计算最高分/最低分 |
| Arrays.copyOf() | 扩展数组添加学生 |
| StringBuilder | 拼接成绩单表格 |
| String.format() | 格式化对齐输出 |
| String.equals() | 按名字查找 |
| String 格式化 | 等级判定与输出 |

---

> 💡 提示：先搭建 while+switch 骨架，逐个实现功能。特别注意 Arrays.copyOf 的用法——它创建新数组，需要把原数组引用指向新数组。
